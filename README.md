# TODO

Aplicação Todo para o curso Tomorrow / UFBA.

## Requisitos

- Java 17
- Maven

## Setup

1. Clone o repositório
2. Navegue até a pasta
3. Instale as dependências: `mvn clean install`

## Uso

Para executar:

`java br.ufba.tomorrow.todo.TodoApplication`
